# Security Policy

## Supported Versions
This project currently supports the `main` branch only.

## Reporting a Vulnerability
Please report security concerns privately to the maintainer before opening a public issue.
Include:
- affected file/component
- impact
- reproduction steps
- suggested mitigation (if available)

## Hardening Notes
- Keep dependencies updated in the virtual environment.
- Use isolated environments only.
- Never commit credentials or private keys.
